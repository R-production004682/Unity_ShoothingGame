### 力を入れたところ  
今回の作品で自分が特に力を入れたところは**2DゲームのAnimationステート遷移の滑らかさ**と、  
**オブジェクトプールを採用した射撃システム**です。
彼女にはアウトゲーム側と、デザインの方を頑張ってもらいました。
